# 감정 VR 웹 애플리케이션

감정을 선택하고 강도를 조절한 후 테트리스 게임을 즐기는 웹 애플리케이션입니다.

## 📁 프로젝트 구조

```
emotion-vr-web/
├── index.html              # 메인 진입점
├── css/
│   └── styles.css         # 전체 스타일
├── js/
│   ├── main.js           # 앱 전체 관리
│   ├── tetris.js         # 테트리스 게임 로직
│   ├── emotionWheel.js   # 감정 룰렛
│   └── intensitySlider.js # 강도 조절
├── assets/
│   └── images/           # 이미지 리소스
├── api/
│   └── backend.js        # 백엔드 연결 준비 (미구현)
├── README.md             # 프로젝트 설명
└── DEVELOPMENT.md        # Copilot 작업 가이드
```

## 🎯 주요 기능

### 1. 감정 선택 (Emotion Wheel)
- **감정 룰렛**: 8가지 감정 중 하나를 룰렛으로 선택
  - 기쁨, 슬픔, 분노, 두려움, 놀람, 혐오, 평온, 흥분
- **애니메이션**: 부드러운 회전 애니메이션과 ease-out 효과
- **색상 구분**: 각 감정마다 고유한 색상으로 시각화

### 2. 강도 조절 (Intensity Slider)
- **슬라이더**: 1~10 단계로 감정의 강도 선택
- **시각적 피드백**: 강도에 따른 색상 그라데이션
- **레벨 표시**: 약함(1-3), 중간(4-7), 강함(8-10)

### 3. 테트리스 게임 (Tetris Game)
- **클래식 테트리스**: 표준 테트리스 규칙
- **7가지 테트로미노**: I, O, T, S, Z, J, L 블록
- **조작법**:
  - ← → : 좌우 이동
  - ↓ : 빠르게 낙하
  - ↑ : 회전
  - Space : 즉시 낙하
- **점수 시스템**:
  - 1줄 클리어: 100점 × 레벨
  - 2줄 클리어: 300점 × 레벨
  - 3줄 클리어: 500점 × 레벨
  - 4줄 클리어: 800점 × 레벨
- **난이도**: 10줄마다 레벨 증가, 속도 빨라짐

## 🚀 시작하기

### 실행 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다.
2. 또는 로컬 서버를 실행합니다:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server 패키지 설치 필요)
npx http-server

# PHP
php -S localhost:8000
```

3. 브라우저에서 `http://localhost:8000` 접속

### 사용 흐름

1. **감정 선택**: "룰렛 돌리기" 버튼을 클릭하여 감정 선택
2. **확인**: 선택된 감정을 확인하고 다음 단계로
3. **강도 조절**: 슬라이더로 감정의 강도 조절
4. **게임 시작**: 테트리스 게임 플레이
5. **재시작**: 언제든지 "다시 시작" 버튼으로 처음부터

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 모던 스타일링, 애니메이션, 그라데이션
- **JavaScript (ES6+)**: 클래스 기반 모듈 구조
- **Canvas API**: 룰렛 및 게임 렌더링

## 📦 모듈 설명

### main.js
- 애플리케이션의 전체 흐름 관리
- 섹션 전환 및 상태 관리
- 이벤트 리스너 설정
- 백엔드 데이터 전송 준비

### emotionWheel.js
- Canvas를 이용한 감정 룰렛 구현
- 회전 애니메이션 처리
- 감정 선택 이벤트 발생

### intensitySlider.js
- 감정 강도 슬라이더 관리
- 실시간 시각적 피드백
- 강도에 따른 색상 변경

### tetris.js
- 완전한 테트리스 게임 로직
- 7가지 테트로미노 구현
- 충돌 감지, 라인 클리어, 점수 계산
- 키보드 컨트롤

### backend.js
- 백엔드 API 통신 준비 (미구현)
- 감정 데이터 전송 함수
- 게임 결과 저장 함수
- 세션 관리 기능

## 🎨 디자인 특징

- **그라데이션 배경**: 보라색 계열의 아름다운 그라데이션
- **카드 디자인**: 흰색 컨테이너에 그림자 효과
- **애니메이션**: 부드러운 페이드인, 슬라이드 효과

- **접근성**: 명확한 색상 대비와 큰 버튼
